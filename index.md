## Projects

### SeedMind
- LLMを前提としない対話AI基盤
- 人格・記憶・会話設計を重視した個人研究プロジェクト
- 設計思想や試行錯誤は Zenn にまとめています

*A conceptual research project focused on designing self-aware, evolving partner agents through architecture, memory, and rules rather than heavy reliance on LLMs.*

---

### Mindra Light
- AI機能を組み込んだ個人向けAIブラウザ
- 縦型タブを採用し、タブ管理と視認性を重視
- テーマ切り替えに対応した柔軟なUI設計
- スプリットビューによる複数ページ同時表示
- 複数のサイトやAIに対して同時に入力・送信できる操作性が特徴

*A personal AI-enabled browser designed to keep thinking and work in one place, with a strong focus on usability, tab management, and multi-view interaction.*

---

### cpp-sudoku-starter
- C++ 学習者向けの数独アプリ教材プロジェクト
- CLI と GUI（raylib）の両方を実装
- 再帰・バックトラックによる数独ソルバーを実装
- CMake を使ったビルド構成とプロジェクト分割
- Git / GitHub を前提とした開発フロー（branch / PR / issue）を解説
- 日本語・英語のドキュメントを用意し、学習用に公開

*A beginner-friendly learning project for C++ and GitHub, designed to teach algorithms and practical development workflow through building a Sudoku application.*
